Informační systém pro školu

Komplexní webová aplikace pro správu žáků, docházky, rozpisů a komunikaci s rodiči. Systém umožňuje učitelům efektivně spravovat třídy a rodiče mají přehled o docházce dětí v reálném čase.

Klíčové funkce:

  • Evidence žáků a docházky
  • Rozvrhy a plánování aktivit
  • Komunikační portál pro rodiče
  • Exporty a statistiky
PHP MySQL JavaScript Bootstrap

Smart Home Monitoring System

Systém pro monitoring teploty, vlhkosti a kvality vzduchu v domácnosti pomocí Raspberry Pi. Data se ukládají do databáze a jsou dostupná přes webové rozhraní s grafy a upozorněními.

Klíčové funkce:

  • Sběr dat ze senzorů DHT22 a MQ-135
  • Webové rozhraní s real-time grafy
  • Emailová upozornění při kritických hodnotách
  • Historická data a trendy
Raspberry Pi Python Flask PostgreSQL

Web pro neziskovou organizaci

Automatizace 3D tiskárny

Raspberry Pi projekt pro vzdálené ovládání a monitoring 3D tiskárny s webkamerou. Umožňuje sledovat průběh tisku odkudkoliv, nahrávat G-code soubory a dostávat notifikace o dokončení tisku.

Klíčové funkce:

  • Vzdálené ovládání přes webové rozhraní
  • Live stream z webkamery
  • Nahrávání G-code souborů
  • Push notifikace při dokončení
Raspberry Pi OctoPrint Python WebRTC
